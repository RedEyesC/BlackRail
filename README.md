#### BlackRail  

一个用于个人测试的unity项目


#### 文件结构  

Engine/Core 核心运行代码   
Engine/GameLogic 游戏逻辑层代码
Editor 编辑器工具代码     
Plugins 第三方库和拓展功能代码 
Init 游戏入口场景
RawData 原始资源  
Asset 游戏运行资源
‌StreamingAssets 用于editor下调试的配置文件



#### 其他

visual studio 使用 CSharpier 进行格式代码
